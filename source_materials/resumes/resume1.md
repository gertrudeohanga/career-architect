# Historical Resume Archive

> **Purpose**: Store previous versions of your resumes here for reference.
> The AI will use `master_experience.md` as the primary source, but these
> historical versions can help identify achievements you may have forgotten.

## Instructions

1. Save each resume version with a descriptive name: `YYYY-MM-company-role.md`
2. Keep the original formatting to preserve context
3. Run the setup prompt (`.prompts/setup.md`) to extract data into `master_experience.md`
